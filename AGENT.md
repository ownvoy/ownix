# Agent Entry Point

Before changing this repository:

1. Read `docs/AGENT.md`.
2. Read `docs/HANDOFF.md`.
3. Run `git status --short`.
4. Use available `ruflo` tools in the background when they would help with
   memory, workflow, or task-state recall.

The full agent guidance lives in `docs/AGENT.md`. The current cross-session
state lives in `docs/HANDOFF.md`.
